# Lua TODO App

Simple TODO application written in Lua.
Currently supports CLI mode. Interactive mode is planned.

## Features

- Add tasks
- List tasks
- Mark/toggle tasks as done
- Remove tasks
- Data stored locally (in Lua table format)

## Run

To run the application just use Lua interpreter:
```bash
lua todo.lua --help
```
or luaJIT:
```bash
luajit todo.lua --help
```
